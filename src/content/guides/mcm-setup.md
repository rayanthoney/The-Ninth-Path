After launching the game for the first time, you must configure the Mod Configuration Menus (MCM) for various mods to match the balanced playstyle of The Ninth Path.

> [!WARNING]
> **Initialize Fully Before Configuration**
> Wait in the starter room for 3 to 5 minutes before opening any menus. Let all scripts initialize and wait for messages saying "MCM registered..." to stop displaying. Setting options mid-registration can break mod scripts.

## Core MCM Configurations

### SkyUI
- **General**: Set Columns to 2.
- **Advanced**: Change font size to Small if you prefer more visible list rows.

### Ordinator
- **Perk Points**: Set to default unless you installed the 20% More Perks option, in which case confirm points are distributed correctly.

### Immersive Armors
- **Distribution**: Uncheck 'Shields' if you experience performance stutters during cell transition.
- **De-leveling**: Keep default settings.

### Wet and Cold
- **Survival**: If using SunHelm or Frostfall, disable freezing and wetness debuffs here to avoid duplicate status effects.
- **NPC Equipment**: Enable hoods and cloaks for NPCs during rain/snow for immersion.
