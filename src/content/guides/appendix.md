This section acts as a glossary and quick-reference index for vocabulary and conventions used in this guide.

## Glossary

| Term | Meaning |
| :--- | :--- |
| **MO2** | Mod Organizer 2, our virtual filesystem mod manager. |
| **SKSE64** | Skyrim Script Extender, allows complex native DLL code execution. |
| **Form 44** | The plugin format version required for Skyrim Special Edition. |
| **Form 43** | Old 32-bit Skyrim Legendary Edition plugin format. Must be resaved in CK. |
| **ESPFE** | A plugin with an ESP extension flagged as ESL, saving an active plugin slot. |
| **BSA** | Bethesda Software Archive. Compressed package containing textures/meshes. |

## Modding Conventions

- **Main Files**: The primary download file of a mod.
- **Update Files**: Small patches released by authors to overwrite parts of the Main file.
- **Optional Files**: Add-ons (e.g. higher-res textures, alternative colors) that are not strictly required but are recommended by this guide.
- **Misc Files**: Utility patches or old versions. Avoid downloading these unless explicitly instructed.
