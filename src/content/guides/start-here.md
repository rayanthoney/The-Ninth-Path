Welcome, Seeker, to the initial step of forging **The Ninth Path**. Before you download a single mod or edit a configuration file, you must understand the rules of this journey. Modding a game to over 1,200 active entries is not a casual weekend task. It is a software engineering exercise that requires focus, exactness, and discipline.

> [!WARNING]
> **Read Before Proceeding**
> Skipping steps, ignoring warnings, or deciding to "wing it" on mod installation choices will result in script lag, save game corruption, and immediate CTD (Crash to Desktop). Follow every word.

## The Philosophy of the list

The Ninth Path is designed to offer a lore-rich, spiritually charged, and high-stakes playthrough. It integrates the entirety of the *Lexy's Legend of the Dragonborn* progression while overlaying custom conflict patches and narrative elements that force tough choices onto the player.

To achieve this:
1. **Never install mods mid-playthrough** unless explicitly marked as safe.
2. **Never change the load order** suggested by LOOT unless a specific custom override is described in this guide.
3. **Never update mods** to newer versions unless you are following a formal release announcement in our Changelog.

## Support Boundaries

We want to help you, but support is strictly limited to the guide as written.

> [!CRITICAL]
> **No Off-Guide Modification Support**
> If you add, remove, or swap even a single plugin from this guide, you have deviated from the path. You are on your own. Do not ask for support in our community channels if you have modified the list.

## Recommended Mindset
- **Expect 10+ hours** of setup time. This is normal.
- **Set aside OneDrive**: OneDrive interferes with Mod Organizer 2's virtual files. Ensure it is disabled.
- **Patience is a resource**: Do not rush. Check each mod version against the version pinned in the guide.
